# Install

```
$ pip install decamelize
```

# Why?

- Turn `CamelCase` to `non_camel_case`
- Clean and focused
- Actively maintained

## Usage

```py
import decamelize

print (decamelize.convert("CamelCase"))
# Eg: CamelCase → camel_case
```
